# GAMEARLY

This repo contains a prototype of the GAMEARLY app, with the Home Screen and stub sections such as Bookmarks, Search, Explore, and Account.

## Run locally
1. Download the repo as ZIP and extract.
2. Double‑click `index.html` to open in your browser (no build tools required).

## Files
- `index.html` — markup + sections
- `styles.css` — dark theme, cards, bottom tab bar, responsive top nav
- `script.js` — hamburger toggle + tiny bookmark demo

## Notes
- Replace placeholder cards with real images/text as needed.

✅ Features

1. Home Screen
	•	Flash Sale
	•	Popular Now 
	•	Recommendations
	•	Recent Releases

2. Search Page
	•	Search bar
	•	Genre filter button
	•	Live search through a small game dataset
	•	Results with bookmark toggle
	•	Tap results to open Game Info page

3. Genre Filter
	•	Checkbox list
	•	Saves selection using localStorage
	•	Applied automatically to Search results

4. Bookmarks Page
	•	Shows all bookmarked games
	•	“Remove” option
	•	Persistent using localStorage

5. Game Info Page
	•	Cover image placeholder
	•	About, Specs, Reviews sections
	•	Bookmark button
	•	Data loaded using query parameters

6. Account Page
	•	Profile picture upload (saved to localStorage)
	•	Username section
	•	Age group + settings using checkboxes
